Una aclaración importante: lo que vamos a hacer a continuación **no debe hacerse**, es simplemente un recurso para explicar cómo funcionan las variables. No lo intentes en tu casa. 

```puppet
function noHacerEstoEnCasa() {
  numerito := 2
  return (numerito)
}
```
